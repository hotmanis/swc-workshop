Virtual Machine

Installation issues can and do happen. To ensure that you can continue to participate in a lesson even if one of your software programs fails, we provide a Linux virtual machine that contains all the necessary software pre-installed. Please install VirtualBox and download this virtual machine image google drive hosted virtual machine image (2.3gigs, if the download is slow, don't worry we will have it on USB sticks at the bootcamp) Load the VM into VirtualBox by doing Import Appliance and loading the .ova file.
